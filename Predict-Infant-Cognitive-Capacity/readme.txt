Title of the Notebook: Group8_Code.ipynb

Brief description of what the notebook does.

Prerequisites
-> create a conda environment with python 3.9.16 using following command:

"conda create --name my_env python == 3.9.16"

-> After that activate the environment

"conda activate my_env"


Installation
-> Install all necessary packages like pandas, numpy, scikit-learn, tensorflow, matplotlib and so on.

-> For causal model, install dowhy package.

-> Command for package install: pip install package_name

Usage
-> Install the nbconvert package by using following command to run the code file from terminal
'conda install nbconvert'

-> After that, use following command
jupyter nbconvert --execute Group8_Code.ipynb

-> Otherwise, you can use cell by cell exection also.
